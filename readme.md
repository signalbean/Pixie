# Pixie

Magisk module that provides Google Photos backup by spoofing device as Pixel XL.

## Features

- Pixel XL device spoofing
- Unlocks unlimited Google Photos backup

## Installation

1. Download zip file
2. Flash via Magisk
3. Reboot
4. Open Google Photos - should show "unlimited backup"
